# BloodBank API

A RESTfull API to manage blood donors,requests, and blood requests

## Overview

This backend application provides endpoints to manage blood donors, track donations, and handle blood requests. It is built with Node.js, Express, and MongoDB.

## Features

- [* ] CRUD operation for donors
- [ ] Track blood availability
- [ ] Handle blood request
- [ ] Authentication
- [* ] Validation and error handling

## Running the project

1. Clone the repository
2. cd blood-bank
3. npm install
4. Create a `.env` file and add PORT=5000
   MONGO_URI=your_mongo_connection_string

5. npm run dev
